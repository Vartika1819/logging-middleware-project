# Logging Middleware Project

## Name: Vartika Singh
## Roll No: RA2311026010371

## Description
This project implements a logging middleware using Node.js.
It fetches notifications, processes them, and sorts them based on priority and time.

## Features
- Logging using external API
- Sorting notifications by weight and timestamp
- Backend implementation using Node.js

## Steps to Run

1. Install dependencies:
npm install

2. Run project:
node index.js

## Output
Top notifications are displayed in sorted order.
